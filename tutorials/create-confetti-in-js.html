Creating a Confetti Effect In Javascript

Creating The Canvas

Creating the canvas in its simplest form can be done by adding the <canvas/> element to your html code.

For the confetti effect to have the most realistic effect we will want the canvas to be the full width and height of the screen. 

// gets the canvas element and assigns it to the canvas  variable for use in js.
const canvas = document.querySelector('canvas');
// gets the drawing context for the canvas to allow drawing to it.
const c = canvas.getContext('2d');

// Sets the canvas width and height to the same width and height as the window.
canvas.width = window.innerWidth;
canvas.height = window.innerHeight;

This still leaves one problem unaddressed and that is that with this canvas being drawn over the entire screen the user wouldnt be able to interact with the website below it. To take care of that we will add the following line of css 

canvas {
	pointer-events: none;
}

Next up we will build the animation loop, so in order to do that it is good to know the basics of drawing in the canvas the following draws a rectangle on the canvas 100pixels from the left, 200 pixels from the top with a width of 50 and a height of 150 

c.fillRect(100, 200, 50, 150)

Setup The Animation Loop

The requestAnimationFrame() method tells the browser that you wish to perform an animation and requests that the browser calls a specified function to update an animation before the next repaint. The method takes a callback as an argument to be invoked before the repaint 

const animate = () => {
	requestAnimationFrame(animate);

	c.clearRect(0, 0, canvas.width, canvas.height);
	c.fillRect(0, 0, 100, 100)

}

animate();
